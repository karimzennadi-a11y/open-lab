## Structure du repo

```
open-lab/
├── cahier-des-charges/   ← spécifications fonctionnelles et techniques
├── plans/                ← schémas, croquis, fichiers CAO (DXF, STEP, SVG)
├── calculs/              ← feuilles de calcul, notebooks Python

├── business-plan/        ← analyse marché, modèle économique

```

## Comment contribuer

1. **Ouvre une Issue** pour signaler un problème, proposer une amélioration, ou poser une question
2. **Fork** le repo, fais tes modifications, ouvre une **Pull Request**
3. Ou envoie un email à [karimzennadi@gmail.com](mailto:karimzennadi@gmail.com) avec l'objet `Contribution Open Lab`

Toutes les contributions sont créditées dans `contributions/`.

### Profils recherchés

- Ingénieurs mécaniques / conception
- Professionnels de santé (ergothérapeutes, infirmiers)
- Designers industriels
- Économistes / analystes marché
- Fabricants / ateliers prototypage
